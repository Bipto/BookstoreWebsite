# Bookstore Website
A clean, modern, and fully responsive web application for browsing, searching, and purchasing books online. This project showcases a full-stack implementation of an online bookstore, including product listings, dynamic search, shopping cart functionality, and an intuitive user experience.<img width="2556" height="1238" alt="Screenshot 2023-08-29 184539" src="https://github.com/user-attachments/assets/b3d69c61-1fb8-4383-9d29-40d41f61559d" />

# Features
- Responsive UI built with mobile-first approach
- Book catalogue with titles, authors, genres and pricing
- Searching and filtering to quickly find books
- Shopping cart with add/remove/update functionality
- User-friendly navigation and clean layout
- Modular, maintainable codebase designed for scalability

# Technologies
- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP
- Database: MySQL
- Tooling: Git, GitHub
